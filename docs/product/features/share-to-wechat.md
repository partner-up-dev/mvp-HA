# 功能：微信内分享

## 用户故事

- 作为发起者，我想在微信内通过菜单分享 PartnerRequest，让对方直接看到分享卡片。

## 流程

- 用户在微信 WebView 打开 PartnerRequest 页面。
- 前端请求后端生成 JS-SDK 签名并完成配置。
- 用户通过微信菜单分享，生成包含标题/缩略图的分享卡片。
- 被分享者点击卡片进入 PartnerRequest 页面。

## 验收标准

- 微信内可正常拉起分享菜单。
- 分享卡片包含正确标题、摘要与缩略图。
- 分享链接可打开对应 PartnerRequest 页面。

## 涉及端

- H5 前端
- 后端
