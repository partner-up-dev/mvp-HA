# 功能：微信内分享

## 用户故事

- 作为发起者，我想在微信内通过菜单分享 PartnerRequest，让对方直接看到分享卡片。
- 作为用户，我希望在首页、创建页、联系作者页也能通过微信菜单分享当前页面，避免只能分享 PR 详情页。

## 流程

- 用户在微信 WebView 打开首页、`/pr/new`、`/contact-author` 或 PartnerRequest 页面。
- 前端请求后端生成 JS-SDK 签名并完成配置。
- 前端调用微信 JS-SDK，同时配置“发送给朋友”和“分享到朋友圈”两种分享数据。
- 用户通过微信菜单分享，生成包含标题/缩略图的分享卡片。
- 被分享者点击卡片后进入对应页面（首页、创建页、联系页或 PartnerRequest 页）。

## 验收标准

- 微信内可正常拉起分享菜单。
- 分享卡片包含正确标题、摘要与缩略图。
- “发送给朋友”和“分享到朋友圈”均可使用。
- 分享链接可打开当前被分享页面（不仅限于 PartnerRequest 页面）。
