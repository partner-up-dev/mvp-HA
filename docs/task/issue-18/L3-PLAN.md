# L3 Plan (Stage 3 Detailed Design)

## Stage 3 Objective
- Add the remaining polish through micro-delight, pacing, and retention support without increasing structural complexity.

## Stage 3 Candidate Work

### 1. Motion & Narrative Rhythm
- Tune section-enter timing and handoff rhythm between Hero, Event Section, and fallback create path.
- Reduce any remaining “component stitched together” feeling in the scroll narrative.

### 2. Return & Retention Support
- Revisit bookmark / return nudges so they support the landing narrative rather than competing with it too early.
- Keep the prompt lightweight and secondary to the two core homepage paths.

### 3. Measurement Refinement
- Add or refine homepage analytics around:
  - Event section impression
  - Highlight card click-through
  - Event plaza continuation
  - Fallback create path entry

### 4. Degraded State Polish
- Re-check empty/error/loading states after the previous stages.
- Ensure even degraded states still feel intentional and not “broken”.

## Exit Criteria
- Homepage feels like a landing page rather than a collection of modules.
- Event participation remains clearly first.
- Community PR creation remains clearly fallback.
- The last 10-20 points come from disciplined details, not heavier structure.
